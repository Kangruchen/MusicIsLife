# 铺面创建指南

## 方法一：在 Godot 编辑器中创建铺面资源（推荐）

### 步骤：

1. **创建 Chart 资源**
   - 在 Godot 编辑器的文件系统面板中，右键点击 `charts/` 文件夹
   - 选择 `新建资源...`
   - 在弹出窗口中搜索 `Chart`
   - 点击 `创建`，命名为你的铺面名称（例如 `hydrogen_chart.tres`）

2. **配置基础信息**
   - 在检查器中设置铺面属性：
	 - `Chart Name`: 铺面名称（例如 "Hydrogen - Easy"）
	 - `Music Path`: 音乐文件路径（例如 "res://music/M.O.O.N. - Hydrogen.mp3"）
	 - `Bpm`: 歌曲的 BPM（例如 128）
	 - `Offset`: 节拍偏移量（秒，默认 0）

3. **添加音符**
   - 展开 `Notes` 数组
   - 点击 `+` 按钮添加新音符
   - 对于每个音符，点击 `<空>` 选择 `新建 Note`
   - 配置音符属性：
	 - `Beat Number`: 节拍编号（支持浮点数，如 1.0、1.5 表示半拍、1.333 表示三连音）
	 - `Beat Time`: 节拍时间（秒，自动计算或手动设置）
- `Type`: 音符类型（0=GUARD, 1=HIT, 2=DODGE）
   
   > **提示**：现在支持细分节拍！可以使用浮点数如 `1.5`（半拍）、`1.333`（三连音）等。详见[细分节拍使用指南](细分节拍使用指南.md)。

4. **保存铺面**
   - 按 `Ctrl+S` 保存资源文件

## 方法二：手动编辑 .tres 文件

可以直接在文本编辑器中创建/编辑 `.tres` 资源文件。参考 `charts/example_chart.tres` 示例。

### 音符类型代码：
- `0` = GUARD（防御） 🛡️（J键）
- `1` = HIT（攻击） ⚔️（I键）
- `2` = DODGE（闪避） 💨（L键）

## 方法三：使用 JSON 格式

已支持从 JSON 文件导入铺面，格式如下：

```json
{
  "chart_name": "Hydrogen - Easy",
  "music_path": "res://music/M.O.O.N. - Hydrogen.mp3",
  "bpm": 128,
  "offset": 0.0,
  "notes": [
	{"beat": 0.0, "type": "HIT"},
	{"beat": 0.5, "type": "GUARD"},  // 支持半拍
	{"beat": 1.0, "type": "DODGE"},
	{"beat": 1.333, "type": "HIT"},  // 支持三连音
	{"beat": 1.667, "type": "GUARD"},
	{"beat": 2.0, "type": "DODGE"}
  ]
}
```

> **注意**：节拍编号 `beat` 支持浮点数，可表示半拍（0.5）、三连音（0.333, 0.667）等。详见[细分节拍使用指南](细分节拍使用指南.md)。

## 在代码中加载铺面

### 方法 1：修改 BeatManager 加载自定义铺面

在 `BeatManager.gd` 中：

```gdscript
# 禁用自动生成测试铺面
@export var generate_test_chart: bool = false
@export var chart_resource: Chart = null  # 拖入你的铺面资源

func _on_music_started() -> void:
	is_playing = true
	current_beat = 0
	next_beat_time = offset
	
	# 加载自定义铺面
	if chart_resource:
		current_chart = chart_resource
		print("已加载铺面: ", current_chart.chart_name)
	elif generate_test_chart:
		_generate_test_chart()
```

### 方法 2：代码中直接加载

```gdscript
var chart = load("res://charts/example_chart.tres") as Chart
current_chart = chart
```

## 铺面设计技巧

1. **BPM 确定**：使用音乐分析工具或在线 BPM 检测器找到歌曲的准确 BPM
2. **Offset 调整**：如果节拍不准，微调 offset 值（通常在 -0.1 到 0.1 秒之间）
3. **音符分布**：
   - 前几拍留空，给玩家准备时间
   - 不要每拍都放音符，留出喘息空间
   - 在音乐高潮处增加音符密度
4. **类型平衡**：合理分配 HIT、GUARD、DODGE 三种类型，避免单一重复

## 测试铺面

1. 在 Godot 场景中选择 `BeatManager` 节点
2. 在检查器中：
   - 取消勾选 `Generate Test Chart`
   - 将你的铺面资源拖到 `Chart Resource` 属性
3. 运行游戏（F5）查看日志输出
4. 根据音乐节奏调整 BPM 和 offset
